# Wikify UI Demo

## Getting Started

## Installation
```bash
yarn install
```

## Usage
create a .`env` file and connect to the backend server, see the [documentation](https://github.com/AshakaE/stego-api) on how to run the backend server

Run the development server:

```bash
yarn dev
```
